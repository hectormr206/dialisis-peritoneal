## 🚀 Getting Started

### 🟡 Pre-requisitos

-   node

### 🔥 Instalación

1. Clonar este repo.
2. Instalar dependencias: `npm install`
3. Iniciar el servidor: `npm start dev`

> Notas: este proyecto corre en: **localhost:3002**

### Producción

1. Una vez instaladas las dependencias, puede realizar la compilación: `npm run build`
